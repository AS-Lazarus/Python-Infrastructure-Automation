# Python-Infrastructure-Automation
This is a collection of Python scripts I developed to automate system log analysis, monitor network health and simplify repetitive infrastructure tasks and configuration auditing.

## 🛠️ Purpose
Coming from an Electrical Engineering background, I view software systems as integrated circuits. These scripts act as "digital probes" to monitor system stability and perform configuration auditing without manual intervention. This dramatically improves system observability through proactive monitoring and automated diagnostic

## 📂 Featured Scripts
*   **`uptime_analyzer.py`**: Performs active reachability checks (TCP) and logs service availability with timestamps for Root Cause Analysis (RCA).
*   **`simple_log_analyzer.py`**: Scans system logs to identify failed login attempts, error patterns,and unauthorized access triggers.
